# 🎙️ Speech Emotion Recognition Project

## Overview
This project detects emotions from speech using deep learning models like Wav2Vec2 and HuBERT.

## Features
- Audio preprocessing
- Emotion detection
- Transcription & summarization

## Tech Stack
Python, PyTorch, Hugging Face, Librosa

## How to Run
pip install -r requirements.txt
python main.py
